Pinterest Image Scraper API

This API allows you to search for random Pinterest images without an API key.

Endpoint

GET /api/search?q=your-search-query

Parameters:

  • q (required): Search query term

Example:

/api/search?q=nature

Response:

{
  "success": true,
  "data": [
    {
      "title": "Image title",
      "description": "Image description",
      "imageUrl": "https://i.pinimg.com/originals/..."
    },
    ...
  ]
}

How to use

Make a GET request to the endpoint with your search query. The API will return a JSON response with matching images from Pinterest.
